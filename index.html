Expected Win Percentage vs Actual

This scatter plot compares the Mets' actual winning percentage (W-L%) against their Pythagorean winning percentage (pythW-L%), which is an estimate of how many games they should have won based on runs scored and runs allowed. Points above the diagonal line indicate the team overperformed relative to their run differential, while points below suggest underperformance.
